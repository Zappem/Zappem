It's easy to connect your application with Zappem.

Composer

				composer require danjohnson95\zappem
			
				composer update

Manual
Download the Zappem PHP class and include it in your files.

Now, in your exception handler, dump this code in there: 

				$Zappem = new \Zappem\Zappem({{project.id}});
				$Zappem->fill($e); //$e is your exception
				$Zappem->send();

If you're using Laravel >=4.2, you can use the Zappem Laravel Connector! 

				composer require danjohnson95\zappem-laravel
			
				composer update

And then stick this code in your Handler.php file: 

				Zappem::send($e);

Barebones
If you want a real commando way of sending your exceptions to Zappem, you just need to send a POST request to
http://localhost:{{port}}/api/v1/exception
